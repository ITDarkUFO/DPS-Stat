# DPS Stat
DPS Stat is a simple Stardew Valley mod that adds a damage per second stat to the bottom of weapon tooltips. Requires SMAPI.

## Compatibility
* Stardew Valley 1.6.14
* Windows/MacOS/Linux
* Multiplayer Compatible
